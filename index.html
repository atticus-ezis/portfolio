Intro

Hello,

 

I'm Atticus, I’m a self-taught programmer who’s interested in web-development. My education is largely project-based, so the purpose of this site is to document and showcase my journey.

 

I started with HTML, CSS, and JavaScript. But have since branched out into frameworks, like Django, apps like Figma (used on this site) and Amazon Web Services.

Projects

Blogs for the Criminally Insane

Generic blog site to learn the basics of Django. Users can login/signup, create, edit and delete their own posts, as well as like, comment and archive public content into folders. Configured and hosted through a virtual server via Amazon Web Services (AWS).
GitHub

The Mix

A personal catalog for my music. Users can shuffle, like and download tracks. Live counters are displayed using a real-time-database in Google Firebase.
GitHub

The Ear Dojo

My first website. A game designed to improve interval recognition for musicians. Users are given 10 questions and if they receive a passing grade they promote a level. Created logic with Javascript and used Amazon S3 for media storage.
