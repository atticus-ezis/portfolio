I’m Atticus,

 

I'm Atticus,I’m a self-taught programmer who’s interested in web-development. My education is largely project-based, so the purpose of this site is to document and showcase my journey.

 

I started with HTML, CSS, and JavaScript. But have since branched out into frameworks, like Django, apps like Figma (used on this site) and Amazon Web Services.

Projects

Blogs for the Criminally Insane

Generic blog site to learn the basics of Django. Users can create , edit and delete their own posts, as well as like , comment , and archive public content into folders . Configured and hosted through a virtual server via Amazon Web Services (AWS).

The Mix

A personal catalog for my music. Users can shuffle, like and download tracks. Live counters are displayed using a real-time-database in Firebase.
GitHub

The Ear Dojo

A game designed to test interval recognition for musicians. Users are given 10 questions and if they receive a passing grade they promote a level. Static storage with Amazon S3.
